# 📽️ Stream Vibe

**Stream Vibe** is a dynamic, user-friendly web application that allows users to discover and explore trending movies and TV shows. The project leverages modern web technologies to deliver a seamless and engaging user experience.

---

## 🚀 Features

- **Trending Content**: Explore the latest movies and TV shows.
- **Search Functionality**: Find movies or TV shows by their title.
- **Dynamic Data Fetching**: Powered by Redux Toolkit for optimized API calls and caching.
- **Responsive Design**: Built with Tailwind CSS to ensure compatibility across all devices.
- **Interactive Carousels**: Built with Swiper JS for showcasing media content.

---

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **API**: [TMDB API](https://www.themoviedb.org/documentation/api)
- **Libraries**:
  - Swiper JS
  - Font Awesome

# 📚 API Reference

This project uses the TMDB API. To set up the API:

1. Create an account on [TMDB](https://www.themoviedb.org/).
2. Generate an API key.
3. Add your API key to the project environment variables:

   ```
   REACT_APP_TMDB_API_KEY=your_api_key_here
   ```

# 🎬 Stream Vibe <a href="https://mustafa-sayed-m.github.io/StreamVibe/">( Live Site )</a>

This is a front-end project built with React to display movies and TV shows using the TMDB API. It features dynamic routing, a search functionality, and detailed information pages for each title. The app is designed with a responsive UI and includes optimized performance through lazy loading and animations.

## 🚀 Tech Stack

- Frontend
- React: JavaScript library for building user interfaces.
- React Redux & Redux Toolkit: State management solution.
- Tailwind CSS: Utility-first CSS framework for styling.
- FontAwesome: Icon library for UI components.
- Formik & Yup: Form handling and validation.
- Swiper: Carousel/slider for creating smooth transitions.
- AOS (Animate on Scroll): Animation library for scroll effects.
- React Lazy Load Image Component: For lazy loading images.

## Backend

- TMDB API: Provides movie and TV show data.

## 📑 Features

- 🎥 Display Movies and TV Shows: Browse through a collection of movies and shows with detailed information.
- 🔍 Search Functionality: Find movies or TV shows by their name.
- 📑 Detailed Information Pages: View additional details about each movie or show.
- 🖼️ Lazy Loading: Optimized loading for images.
- 🌐 Responsive Design: Compatible with all device sizes.
- 🎨 Smooth Animations: Engage users with modern animations and scroll effects.

## 📂 Project Setup

To set up and run the project locally, follow these steps:

1. Clone the repository:

   ```
   git clone https://github.com/your-username/movies-shows-app.git
   ```

2. Navigate to the project directory:

   ```
   cd movies-shows-app
   ```

3. Install dependencies:

   ```
   npm install
   ```

4. Start the development server:

   ```
   npm start
   ```

5. Visit the application: Open your browser and go to [here](http://localhost:3000 "Open Local Host").
